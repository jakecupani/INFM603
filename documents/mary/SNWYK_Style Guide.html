The header above is a mix of a nav class [nav class="navbar navbar-dark bg-dark"] and an order list [ol class="breadcrumb"]. All pages will use the default container [div class="container"] class.

The site will have a big enter button [class="btn btn-secondary btn-lg"] to get you in the door.

"Secondary" in the class above refers to a ranking scheme, which translates to a color scheme. I think gray makes the most sense for our predominant color scheme. Individual buttons each have a class [class="btn btn-XXX"], where XXX corresponds to the label (lowercase) on the button below.

The button toolbar class [class="btn-toolbar"] may be handy for the buttons on the quiz pages for selecting options, such as easy/hard or a list of individual space topics. The class allows for groupings.

Use the paragraph tag, not the div tag, if you want some white space around your text. Otherwise everthing gets pretty crammed together. Note that the paragraph tag provides little to no white space at the top; white space provided at the bottom. If you look at the html for this page, you'll see I inserted empty paragraphs to make white space above some paragraphs.

Use this paragraph class [p class="lead"] if you want your paragraph to stand out a bit, but not so much that people think it's another header.
